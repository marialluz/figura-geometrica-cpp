Figura Geométrica - Estruturas avançadas

Bem-vindo à documentação do projeto de Figuras Geométricas. Este documento fornece uma visão abrangente das classes e funcionalidades desenvolvidas para manipular figuras geométricas simples, realizando operações de desenho em uma matriz de pontos. O projeto visa criar uma estrutura flexível e reutilizável para a representação e manipulação de figuras geométricas em um ambiente tridimensional.

O projeto inclui a criação de uma classe abstrata chamada FiguraGeometrica, que serve como base para várias outras classes concretas. Cada classe concreta representa um tipo específico de figura geométrica e implementa a funcionalidade de desenho ou remoção. Além disso, o projeto permite a leitura de instruções de desenho a partir de um arquivo, gerando representações gráficas com base nessas instruções.

Este documento detalha a estrutura das classes, métodos e seu uso, bem como fornece exemplos de como realizar operações de desenho e remoção de figuras geométricas. A documentação visa auxiliar desenvolvedores e usuários a compreender e utilizar as funcionalidades do projeto de forma eficaz.
